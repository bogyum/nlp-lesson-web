Jupyter Lab 설치 및 설정

1단계: 시스템 업데이트

먼저 Ubuntu 시스템을 최신 상태로 업데이트합니다.

시스템 업데이트
sudo apt update
sudo apt upgrade -y

2단계: Python 및 pip 설치

Python과 pip가 설치되어 있지 않다면 설치합니다.

Python 및 pip 설치
sudo apt install python3 python3-pip python3-venv -y
python3 --version
pip3 --version

3단계: 가상환경 생성

프로젝트별로 독립적인 환경을 만들기 위해 가상환경을 생성합니다.

가상환경 생성 및 활성화
python3 -m venv jupyter_env
source jupyter_env/bin/activate

가상환경의 장점

  • • 프로젝트별로 독립적인 라이브러리 관리
  • • 시스템 Python 환경과 분리
  • • 다른 프로젝트와의 충돌 방지

4단계: Jupyter Lab 설치

가상환경이 활성화된 상태에서 Jupyter Lab을 설치합니다.

Jupyter Lab 설치
pip install jupyterlab
jupyter lab --version

5단계: Jupyter 설정 파일 생성

Jupyter Lab의 설정 파일을 생성합니다.

설정 파일 생성
jupyter lab --generate-config
ls -la ~/.jupyter/

6단계: 비밀번호 설정

Jupyter Lab에 접속할 때 사용할 비밀번호를 설정합니다.

비밀번호 설정
jupyter lab password
# 비밀번호를 입력하라는 프롬프트가 나타납니다
# 예: mypassword123

7단계: 외부 접속을 위한 설정

외부에서 접속할 수 있도록 설정을 수정합니다.

설정 파일 편집:

설정 파일 편집
nano ~/.jupyter/jupyter_lab_config.py

다음 내용을 추가:

설정 내용
# 외부 접속 허용
c.ServerApp.ip = '0.0.0.0'

# 포트 설정 (기본값: 8888)
c.ServerApp.port = 8888

# 브라우저 자동 열기 비활성화
c.ServerApp.open_browser = False

# 토큰 인증 비활성화 (비밀번호만 사용)
c.ServerApp.token = ''

# 허용된 원본 설정
c.ServerApp.allow_origin = '*'

# CORS 설정
c.ServerApp.allow_credentials = True

8단계: 방화벽 설정

Ubuntu 방화벽에서 Jupyter Lab 포트를 열어줍니다.

방화벽 설정
# UFW 방화벽 활성화 (필요한 경우)
sudo ufw enable

# Jupyter Lab 포트 허용
sudo ufw allow 8888

# 방화벽 상태 확인
sudo ufw status

9단계: Jupyter Lab 시작

설정이 완료되면 Jupyter Lab을 시작합니다.

Jupyter Lab 시작
jupyter lab

접속 방법

로컬 접속: http://localhost:8888

외부 접속: http://서버IP:8888

예시: http://192.168.1.100:8888

포트 번호 변경 방법

기본 포트(8888) 대신 다른 포트를 사용하고 싶다면:

1. 설정 파일에서 포트 변경:

포트 변경
# ~/.jupyter/jupyter_lab_config.py 파일에서
c.ServerApp.port = 9999  # 원하는 포트 번호로 변경

2. 방화벽에서 새 포트 허용:

방화벽 포트 허용
sudo ufw allow 9999  # 새 포트 번호
sudo ufw status

3. 특정 포트로 시작:

특정 포트로 시작
jupyter lab --port=9999

보안 주의사항

  • • 강력한 비밀번호 사용: 예측하기 어려운 비밀번호를 설정하세요
  • • 방화벽 설정: 필요한 포트만 열어두세요
  • • HTTPS 사용: 프로덕션 환경에서는 SSL 인증서를 사용하세요
  • • 정기적인 업데이트: Jupyter Lab과 관련 패키지를 정기적으로 업데이트하세요
